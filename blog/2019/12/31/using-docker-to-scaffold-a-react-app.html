Using docker to scaffold a React App

blog docker

I am relearning react. I also installed docker to stop myself from installing other softwares on my laptop. So here i use docker to build a container which i can use to build a container with scaffolding for developing a react app.

I have the nessacery files on git gist, so here goes:

git clone https://gist.github.com/Trshant/2cc6cf99749c979b8a1be5e39b45f757 react_docker

Think up a nice name for a project. I am going to call it learn-react

Then change directories and set up the directory for the actual code.

cd react_docker
mkdir code
sudo chmod 777 code
sed -i -- 's/<project-name>/learn-react/g' *

Now we build the docker container with the create-react-app

docker-compose build create_env

Once thats done.

docker-compose run create_env create-react-app learn-react

After doing this, we have the scaffolding for a new react app ready in the code folder.

We can now run the actual docker to run the container with the new react app.

docker-compose up learn-react --build

Congratulations! we can now learn react without having installed anything extra into our computer.
